Trucker Repair App 

A React-based mobile application designed to help truck drivers quickly find nearby repair shops and services while on the road.

Purpose
This app addresses the critical need for truck drivers to locate reliable repair services during emergencies or routine maintenance. It provides location-based search, detailed shop information, and trucker-specific services.

Features
Location-based search - Find repair shops near your current location
Detailed shop information - View ratings, hours, services, and contact details
Trucker-specific filtering - Find shops that accommodate large vehicles
Service categorization - Filter by Engine, Brakes, Tires, Electrical, 24/7 services
Interactive map - Visual representation of nearby shops
Mobile-first design - Optimized for use on the road

Tech Stack
Frontend: React

Styling: Tailwind CSS

Icons: Lucide React

Maps: Google Maps API integration

Data: Google Places API for shop information (planned)
